# 🔬 Alliance Technology Complete Guide

> **Alliance Tech provides permanent, account-wide passive buffs.** Maxing the right ones gives you a massive edge.

---

## Tech Tree Priority

| Priority | Tech | Effect | Why First |
|:---:|------|--------|-----------|
| 🥇 1 | **Territory Expansion** | More alliance territory | Benefits EVERY member |
| 🥈 2 | **Attack** | +ATK% for all troops | Direct combat power |
| 🥉 3 | **Defense** | +DEF% for all troops | Survivability |
| 4 | **HP** | +HP% for all troops | Universal tankiness |
| 5 | **Gathering Speed** | Faster resource collection | More resources for upgrades |
| 6 | **Construction Speed** | Faster building | Speeds up HQ rush |
| 7 | **Research Speed** | Faster research | Compounds with other buffs |


## Territory Expansion — The Hidden Power

| Level | Effect |
|:---:|--------|
| 1-5 | Basic territory |
| 6-10 | More resource tiles, better gathering |
| 11-15 | Strategic positioning for wars |
| 16+ | Dominance on server map |

<div class="callout tip">
<div class="callout-title">🏰 Territory > Combat Tech (Early Game)</div>
More territory = more resources = faster growth = higher HQ = more power. In the early game, territory expansion benefits your entire alliance more than +5% ATK.
</div>
