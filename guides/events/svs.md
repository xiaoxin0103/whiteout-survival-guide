# 🌍 Server vs Server (SvS) — Complete Guide

> **The biggest event in Whiteout Survival.** Your server fights another server. Winner takes glory, rewards, and bragging rights.

---

## What Is SvS?

| Detail | Info |
|--------|------|
| **Type** | Cross-server Alliance War |
| **Duration** | Multi-day event |
| **Participants** | Entire server vs another server |
| **Win Condition** | Highest total points across all phases |
| **Top Reward** | Exclusive hero shards + massive gems + server-wide buffs |

---

## SvS Phases

| Phase | Duration | What Happens |
|-------|----------|-------------|
| **Prep** | 24-48h | Stockpile resources, speedups, attack boosts |
| **Battle** | 2-3 days | Open PvP between servers. Attack enemy alliances. |
| **Settlement** | 24h | Points tallied. Rewards distributed. |

## Preparation Checklist

- [ ] Stockpile 500+ hours of speedups
- [ ] Max hospital capacity (you WILL take losses)
- [ ] Coordinate with top 3 alliances on your server
- [ ] Set up shared Discord for cross-alliance communication
- [ ] Save attack boosts and march capacity items
- [ ] Ensure all heroes are healed and geared

### F2P SvS Strategy

<div class="callout tip">
<div class="callout-title">🎯 F2P SvS Rule</div>
Don't try to fight whales head-on. Focus on raiding resources, killing troops, and supporting stronger allies. <strong>Points > kills.</strong> You can score top 10 without winning a single duel.
</div>

## Server-Wide Strategy

### Before SvS

| Action | Owner |
|--------|-------|
| Negotiate temporary server-wide truce | Top 5 Alliance R5s |
| Assign war targets | Alliance leaders |
| Share resource/speedup stockpile status | All members |

### During SvS

| Phase | Action |
|-------|--------|
| **First 6 hours** | Scout enemy server. Find weakest alliances. |
| **Day 1** | Hit undefended alliances. Farm points. |
| **Day 2** | Coordinated attacks on enemy top alliances. |
| **Last 6 hours** | All-out push. Burn all saved items. |

<div class="callout warning">
<div class="callout-title">⚠️ Don't Waste Points Early</div>
Points in the last 6 hours count the same as points in the first hour. Save your strongest attacks for the final push when the enemy is exhausted.
</div>

---

> **After SvS:** Heal all troops immediately. Don't walk around with wounded — the post-SvS cleanup phase is when scavengers hunt easy targets.
