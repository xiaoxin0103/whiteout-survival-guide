## The Golden Rule of Events

> **Save speedups and resources for event days.** Don't use them the moment you get them.

| Event | What to Save |
|-------|-------------|
| Arms Race | Speedups, hero shards, gear materials |
| Alliance Duel | Resources, construction completions |
| King of Icefield | Attack boosts, march capacity items |
| Seasonal Events | Everything — seasonal rewards are the best in the game |
