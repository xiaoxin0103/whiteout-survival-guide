# 💰 Pack Value Analysis — What's Worth Buying

> **Not all packs are equal.** Some give you 10× the value of others. This guide ranks every purchase by gems-per-dollar efficiency.

---

## The Value Formula

```
Pack Value = (Gems + Resource Value + Speedup Value + Shard Value) ÷ Price
```

Resources and speedups are converted to their gem equivalent for fair comparison.

## Before You Buy — The 24-Hour Rule

If you see a pack and think "I need this" — wait 24 hours. If you still want it tomorrow, buy it. Most "limited time" packs are designed to trigger impulse purchases. The 24-hour rule kills FOMO.
